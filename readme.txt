Plugin for automated data migration from Joomla K2 to WordPress. Convert Joomla K2 to WordPress easily and fast.


== Description ==
You have chosen Joomla K2 to be a platform to run your website on. But then you’ve decided that WordPress might be a better CMS for your site.  Now you are wondering how to move your data from Joomla K2 to WordPress. Furthermore, being a busy person or lacking coding skills may cause some difficulties in website migration process which you would like to avoid. To transfer your data from Joomla K2 to WordPress accurately, flawlessly and effortlessly, consider to use this plugin.

= HOW THIS PLUGIN HELPS: =
Possibility to preview the future results via running the Free Demo Migration;
The approximate time of migration is 15 minutes;
Ability to indicate the content items you want to move from Joomla K2 to WordPress CMS (Data “Mapping” option);
The pricing depends on the amount of content you’re going to move.

= WHAT WILL BE MOVED FROM JOOMLA K2 TO WORDPRESS? =
* Articles with date of creation, author, status included;
* Content and all media files;
* Users with all the components such as user created date, nickname, first and last name;
* Attachments (images and videos);
* Categories;
* Tags;
* Menu;
* Images and featured images;
* Comments with content, date of creation, status and author.

= PRESERVED SEO JUICE AFTER THE MIGRATION =
* Migration of keywords, metadata/meta description;
* SEO aliases and SEO friendly URLs migration.

= COMPATIBLE WITH: WORDPRESS 4.1.X – 5.0.X. =
Availability of live help. If you have any migration related questions, our tech team will try to answer them.
If you want to perform a customized migration, make a request for a CMS2CMS data migration service package.

== Installation ==
1. Download the plugin zip file.
2. Go to the Admin Panel -> Plugins -> Add New -> Upload Plugin.
3. Choose file in zip format and click the “Install Now” button.
4. Go to Admin -> Plugins, activate the “CMS2CMS: Automated Joomla K2 To WordPress Migration Plugin“.
5. Register an account or sign in to the existing one.
6. Provide your Existing site URL address.
7. Choose between the installation of the Connector Extension or the Connection Bridge. You can find a detailed instruction [here](https://cms2cms.com/faqs/what-is-the-connection-bridge-and-how-to-install-it/).
8. Go back to the plugin and click “Verify Connection”. You’ll be redirected to the CMS2CMS website to proceed with your migration.
